末世逃亡时，还能享受美食，而不是饿死！🍟

源自这样的设定：我记得有企业说，在地球毁灭时，每日发送1000艘太空飞船把100万人迁往火星。如此我当真思考过，设想其中会遇上的各样问题，最后找到根本关键处，应让这批身处其中的人先吃好，再提其它，只要活着都不是问题。毕竟，那里是全人类粮食与人口矛盾最尖锐的地区。计算后的思路是，1年以内的旅程，把飞船制造为可食用性；一年以上，🐔还是靠生态球模式的自给自足生产吧。

事实该是认真的，不管模式怎样，在人体冬眠技术尚未成熟的时代，如何通过综合方案以技术提升，让宇航舱尽可能多地携带粮食、药品出征远方呢？除了在有限空间里开展畜牧、种植等等自给自足方案，采取的额外路径必须是：尽可能地在建造初期，把航天器内部施以可食用化，融入飞船结构之中，让人类在宇宙征途中，👩‍🚀可以走得更远一点、更宽裕一点。

人在紧急情况时可以熬制马鞍、皮鞋👞、皮带用作蛋白质的应急补充；在无土的地方把液体的凝固物制作成砖块建设坚固的房屋；于急行军的路上携带坚硬但热值含量高的面制品作为食品；古代战争年中百姓蒸煮米浆筑为城墙，当饥荒来到时拯救了数千条人命。这些是人类最奇特的设计，却也是危急时候宇航员生存的保证，这窘境开辟了饮食领域的新品类。

在逃亡路上的沿途天体设立美食餐厅和飞船修补站，是不是很米其林。配方从何而来，哪些人愿意首先尝鲜？不用想当然地认为是事业开创的机会莅临了——未来的故事才刚开始，是乐队篇章的第一声弦音🎵。我自然十分兴趣开辟一项非代码类开源社区，让美食爱好者热衷于把地球食材加工与烹饪技法贡献出来，分享在不同星球环境下可能出现的各种美拉德效应，来丰富人们味蕾。这其间包含昆虫学家、理化药学专家、分子料理师、厨师、营养师以及机械工程师等集体者智慧，动用到一切可让物品逆向还原为美食的工程技术，形成理论公式。🗝验证呢？为吃货在地球上先试点做些太空食物品味馆怎么样？替宇航员们先尝尝，筛选出最优口味和美食密码。但我想，还是回归地球现实，先关注碳减排，让人类末日指针向后先退几秒，替世界再争取一下吧。

![可食用太空船](https://github.com/ReadClaw/StarShip/raw/main/StarShip/starship.jpg)

相较航天器上的大物件，作为日常用的小物品是个初始挑战。环保主义者的眼里容不得砂子，世界雨林🌲不断遭受破坏，一年消耗390亿双的一次性木筷像780亿支穿心箭一般射中人心。消费是高频次的，使用者轻描淡写，而林木面积持续减少着。不过，一次性木筷是个好东西，它方便、隔绝传染、便宜、正规生产的话还算干净。虽然知晓制造它需砍伐森林、动物流失🐦、水土变质，但人们照样使用，无需芥蒂，碳信用生而原罪。只是水无定势，物无常形，多年前人人厌弃的地沟油现如今已经变为世界抢手的生态油。你能确认下一个改变世界的是一次性筷子，就和你的Claw一起往前走，属于另一个圾垃回收油变身时刻。
